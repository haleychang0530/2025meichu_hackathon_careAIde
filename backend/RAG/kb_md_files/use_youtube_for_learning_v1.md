## 在 YouTube 找教學影片（簡單版）

1. 打開【YouTube】App 或網站。  
2. 在搜尋框輸入你想學的東西（例如：手機教學、料理）。  
3. 挑觀看數多、頻道名稱可靠的影片。  
4. 點開就能邊看邊學。  

👉 小提醒：遇到太誇張的廣告影片，要小心可能是詐騙。
