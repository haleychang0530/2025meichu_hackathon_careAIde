## Windows 11 放大字體（簡單版）

1. 點左下角【開始】→【設定】。  
2. 選【輔助工具】→【文字大小】。  
3. 拉動滑桿，把字變大。  
4. 按【套用】，字體就會放大。  
5. 可以試著逐一調整，找到最舒服的大小。
