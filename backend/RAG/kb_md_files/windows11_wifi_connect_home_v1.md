## Windows 11 連上 Wi-Fi（簡單版）

1. 點右下角【網路圖示】（像小地球或波浪）。  
2. 選家裡 Wi-Fi 名稱，按【連線】。  
3. 輸入 Wi-Fi 密碼。  
4. 密碼分大小寫，打錯會連不上。
